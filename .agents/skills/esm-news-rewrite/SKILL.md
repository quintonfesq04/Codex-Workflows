# ESM News Rewrite

## Trigger Examples

- `Give me my own change so I don't copy SNY's tweet`
- `Rewrite this Mets news for X, source is @Mets`
- `Make this sound like us`

## Workflow

1. Identify source facts versus user commentary.
2. Preserve facts, names, dates, and attribution.
3. Substantially rephrase wording, structure, hook, and framing.
4. Keep the post in ESM social style.
5. Add source attribution when the input is reporting from another account or outlet.
6. Keep copy under the requested platform limit.

## Verification

- No copied sentence structure from source.
- Source attribution included when needed.
- Current date and sports context verified if the news is time-sensitive.
- Character count checked when for X.

## Expected Output

- One clean rewritten post in chat.
- Optional alternate version when useful.
