# List And Ranking Carousel Format

Use this preset for "ball knowers," top lists, underrated players, rankings, watch lists, and multi-player carousels with a repeated slide pattern.

## Trigger Examples

- `Format: list`
- `Make a Ball Knowers Only carousel`
- `Top 5 under-the-radar pitchers`
- `Make a ranking carousel for these players`

## Best For

- Multiple players or teams.
- One repeated item format per slide.
- Posts where the viewer should swipe for the next name.
- Lists that should be easy to scan quickly.

## Default Slide Map

1. Cover: list promise, often two or three stacked lines.
2. Item 1: player/topic, short argument, bottom stat.
3. Item 2: same structure.
4. Item 3: same structure.
5. Item 4: same structure.
6. Item 5: same structure.
7. Optional debate or meme slide when the list needs a closing reaction.

## Aesthetic Direction

- Repetition is the strength. Each item slide should feel like part of the same set.
- Keep the top copy band and bottom stat treatment consistent.
- Use short copy. Do not let list slides become essays.
- Player image should dominate the lower section.
- If numbering is used, keep it clean and secondary. The player/topic and stat should still carry the slide.
- Cover may use league colors or a neutral MLB palette when the post is not team-specific.

## Required Inputs

- Format label or list title.
- Team/league colors.
- Cover image.
- One image per list item.
- Item names, copy, and bottom stat lines.
- Optional final meme/callout slide.

## Verification

- Every item slide follows the same visual hierarchy.
- No image is repeated unless the user explicitly requests it.
- Slide order matches the supplied list order.
- Cover sets up the list clearly without subtext.
- The repeated stat treatment is readable on every slide.
