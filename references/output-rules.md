# Output Rules

## Social Posts

- Keep tweet/X copy in chat.
- Keep posts under the Free X character limit.
- Use stat bullets and a direct question when the goal is replies.
- Match clips to the stat window.
- Do not include research links inside social copy.
- Do not overhype weak statistical cases.

## Instagram Carousels

- Export final rendered slides as 1080x1350 PNGs.
- Save each post in a dedicated folder with clear names such as `01_cover.png`, `02_slide.png`, and `06_meme.png`.
- Mirror completed local carousel folders into the user's `Documents` folder when producing assets on their machine.
- Use supplied images in the exact order provided unless the user explicitly changes the order.
- Include one meme still slide unless the user explicitly says not to.
- Ignore meme videos for carousel slides.
- Do not verify stats that the user already supplied as final copy.

## Video

- Default MLB compilation ratio: 16:9.
- Render at 1920x1080.
- Minimum duration: 45 seconds.
- Maximum duration: 3 minutes.
- Use blended transitions.
- Use Baseball Savant clips first.
- Use MLB.com highlights only when Savant does not produce usable clips.
- For balanced or skeptical pitcher posts, include both highlights and lowlights.

## Verification

- Verify current date and schedule before using `today`, `tonight`, or `tomorrow`.
- Verify probable starters before asking users to predict a final pitching line.
- Verify ranking source and filter before publishing ranks.
- Run dry-runs before long renders when possible.
